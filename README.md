# Employee Attrition Prediction using Machine Learning

##  Project Overview

This project aims to predict employee attrition using machine learning techniques. The dataset was explored, cleaned, preprocessed, and used to train multiple classification models. The performance of each model was evaluated using standard classification metrics.

---

##  Objective

The objective of this project is to identify employees who are likely to leave the company by analyzing various employee-related factors such as age, job role, income, work-life balance, and years at the company.

---

##  Dataset

- **Dataset:** IBM HR Analytics Employee Attrition Dataset
- **Records:** 1470
- **Features:** 35
- **Target Variable:** Attrition

##  Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score

---

##  Results

Among the three models, **Logistic Regression** achieved the best overall performance with an accuracy of approximately **86%**, making it the most suitable model for predicting employee attrition.
